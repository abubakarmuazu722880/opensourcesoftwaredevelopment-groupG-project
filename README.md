# 🌐 Personal Portfolio Website

A clean, responsive, and modern personal portfolio website template built with **HTML5**, **CSS3**, and **Vanilla JavaScript**.

This project was developed as a **group open-source project** to practice collaborative software development using Git and GitHub.

## 🚀 Features

- ✅ Fully responsive (mobile, tablet, desktop)
- ✅ Sticky navigation bar with smooth scrolling
- ✅ Hero section with call-to-action buttons
- ✅ About Me section with downloadable CV link
- ✅ Animated skill progress bars
- ✅ Projects showcase section
- ✅ Contact form with validation
- ✅ Social media links
- ✅ Clean, modern UI with consistent design tokens

## ⚙️ How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR-USERNAME/portfolio-website.git
   ```

2. **Navigate into the project folder:**
   ```bash
   cd portfolio-website
   ```

3. **Open `index.html` in your browser:**
   - Simply double-click `index.html`, OR
   - Use a VS Code extension like **Live Server** for auto-reload

## 🔱 How to Contribute (Group Members)

### Step 1 — Fork the Repository
Click the **Fork** button at the top-right of this repository page to create your own copy.

### Step 2 — Clone Your Fork
```bash
git clone https://github.com/YOUR-USERNAME/portfolio-website.git
cd portfolio-website
```

### Step 3 — Create Your Own Branch
Use your name or assigned section as the branch name:
```bash
git checkout -b feature/your-name-section
```
> Example: `git checkout -b feature/amina-skills-section`

### Step 4 — Make Your Changes
Edit the assigned section of the project. See the **Task Assignment** table below.

### Step 5 — Commit and Push
```bash
git add .
git commit -m "Add: [describe what you changed]"
git push origin feature/your-name-section
```

### Step 6 — Open a Pull Request (PR)
- Go to the **original** repository on GitHub
- Click **"Compare & pull request"**
- Write a clear description of your changes
- Submit the PR for the group leader to review
